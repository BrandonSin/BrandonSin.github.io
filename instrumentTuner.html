Violin Tuner Report


A Violin Tuner App


This project was made with React frontend. The purpose of this project was to incorporate audio without voice command. From there I had several ideas pertaining to what the App could become.




From all my rough sketches, I wanted to do something that is more practical than a gaming application. I went with an instrument tuning app especially for violin because I always needed a tuning device back when I played the violin. On top of that it is a practical application with useful features.





These are some of ideas for the interfaces of the tuner. I decided to go with sketch 4 as I believe I could incorporate the idea within the time and ability constraint. In addition, the application would provide the user the frequency and the current note which are useful for tuning any instrument.


Artistic Choices

  • Background Colors are similar to a varnished Violin
  • The note changes to color green whenever the sound is within the frequency range
  • The meter gives a visual to the user where the current frequency is
